# PoundsTo---EmbeddedFinal-Project---WateringSystem
The purpose of this project is to monitor the moisture level of the soil that a plant is in, and if the level is too low it will add water to the soil until the moisture is above a specified acceptabled threshold. Users can also view their plantâ€™s moisture level over the internet with Thingspeak.

## Components
- [x] Capacitive soil moisture sensor (ADC A1: P1.1)
- [x] Temperature sensor (ADC A12)
- [x] ESP-01 WiFi module (UART: P4.2, P4.3)
- [x] Micro servo (Timer_B3 PWM: P6.3)
- [x] Red LED (P1.0)
- [x] Green LED (P6.6)
- [x] Push button (moisture threshold set) (P2.3)
- [x] Push button (water level reset) (P4.1)
